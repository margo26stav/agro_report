# 🌾 Агро Бот

Telegram-бот для структурирования отчётов о полевых работах.

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `ANTHROPIC_API_KEY` | Ключ от console.anthropic.com |

## Деплой на Railway

1. Загрузи файлы на GitHub
2. Зайди на railway.app → New Project → Deploy from GitHub
3. Добавь переменные окружения в Settings → Variables
4. Готово!
